# PlaceX - Run Commands for VS Code

## 🚀 **QUICK START COMMANDS**

### **Option 1: VS Code Debug (Recommended)**
1. Open VS Code
2. Press **F5** → Select "Start Full App (Mock)"
3. Application starts automatically

### **Option 2: VS Code Tasks**
1. Open VS Code
2. **Ctrl+Shift+P** → Type "Tasks: Start Full Application"
3. Application starts automatically

### **Option 3: Terminal Commands**
```bash
# Start Backend (Terminal 1)
cd src/server-side
node index-mock.js

# Start Frontend (Terminal 2)
cd ../../
npm start
```
